# Lithology Pattern Palettes

Built-in lithology SVG patterns are seeded from:

- Upstream repository: `https://github.com/equinor/lithology-patterns`
- License: MIT
- Source asset folder: `assets/svg`
- Local source checkout: `repos/lithology-patterns`
- Runtime seed snapshot: `app/src/subsidence/data/dictionaries/lithology_patterns/equinor`

The application does not depend on the local source checkout at runtime. The
checkout is a reference/source snapshot; project databases are seeded from the
runtime seed snapshot under `app/src/subsidence/data/dictionaries`.
